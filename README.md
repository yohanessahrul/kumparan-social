## Run Locally

Clone the project

```bash
  git clone https://github.com/yohanessahrul/kumparan-social.git
```

Go to the project directory

```bash
  cd kumparan-social
```

Install dependencies

```bash
  npm install sass
```

```bash
  npm install
```

Start the server

```bash
  npm start
```

  
## Tech Stack

**Client:** React, Redux, Sass
